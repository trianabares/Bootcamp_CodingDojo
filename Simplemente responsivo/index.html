@media queries

CSS uses the @ media tule to detect information about the device veing used to access the website. For exaple, it can be used to detect the size of the divice, its capailities, and its orientatios. For this course, we will be using it to detect the device size, but keep in mind thar is is capable of much more

Desktop: Width greater than or equal to 1024px

Tablet: Width between 481px and 1023px

Phone: Width less than or equal to 480px
